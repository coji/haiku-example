claude3 haiku を使って、google 検索経由でウェブサイトを読み取り、企業の業種を判別させます。

実行例
```sh
pnpm tsx main.ts    
URL: https://www.tis.co.jp/company/information/
このWebページの内容から判断すると、TIS株式会社は情報サービス業界に属する企業だと考えられます。

根拠は以下の通りです:

- 会社概要に記載されている通り、TIS株式会社は1971年に創業され、2008年に設立された企業です。
- 主な事業内容は情報処理サービス、システム開発、ITコンサルティングといった情報サービス業に関連する業務です。
- 従業員数が約2万2千人、売上高が約5,084億円と、規模の大きな情報サービス企業であることが分かります。
- 東京証券取引所のプライム市場に上場しており、主要な情報サービス企業の1つであると言えます。

以上から、TIS株式会社は情報サービス業界に属する大手企業であると判断できます。

URL: https://doda.jp/companyinfo/
基づいて、このWebページの内容から会社の業種を判断すると、IT・通信業界およびインターネット・広告・メディア業界と思われます。

理由は以下の通りです:

1. Webページのコンテンツ自体がdoda (デューダ)という転職サイトのものであり、IT・通信やインターネットサービスに関連する業界の情報を提供しているためです。

2. Webページ中にGoogle Tag Managerの導入コードが使われていることから、WEBサイトやWebアプリケーション運営を行っている企業の特徴がうかがえます。

3. 求人情報の検索や転職支援サービスなどが主要な機能であることから、IT・通信やインターネット・メディア関連の業界を対象としていると考えられます。

以上より、このWebページの運営企業は主にIT・通信業界やインターネット・広告・メディア業界に分類されると判断できます。

URL: https://employment.en-japan.com/tenshoku-daijiten/13194/
会社の業種を判別するのは難しいですが、以下の情報から考えられることがあります:

- 会社概要に記載されている事業内容から判断すると、この会社は転職情報サイトを運営している会社であると思われます。
- エン・ジャパンという会社名から、IT/Web関連の業界に属していると考えられます。

具体的には、「インターネットサービス」や「メディア・マスコミ」などの業種が考えられます。
転職情報サイトの運営は主要な事業内容のようですが、関連事業や子会社の情報もあるため、より詳細な業種の判断は難しいと言えます。
会社概要を詳しく見ても、この会社の主たる業種を特定するのは困難ですが、IT/Web関連の業界に属する企業であると推測されます。

URL: https://webmo.co.jp/cl2011f/

以上から、この会社は主にWebサイトやシステムの制作・開発を行う IT/Web 制作会社であると分析できます。
```
